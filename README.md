# ChatGPT-CLI

![mio256](https://avatars.githubusercontent.com/u/71450182)

## Overview

Simple CLI made with Go for ChatGPT.

## Requirement

- [go-chatgpt](https://pkg.go.dev/github.com/ayush6624/go-chatgpt@v0.2.0)

## Usage

```sh
# set `OPENAI_API_KEY = ******`
$ vim .env

$ go build -o a.out
$ ./a.out
```
